# Inventory System

> Version: 0.1

A Custom inventory system, designed to be used easily, without any issues.

# Installation

1. Download the `Inventory.rbxmx` file.
2. Drag and drop it in your Roblox game's `workspace`.
3. Move the `Inventory (ScreenGui)` into `StarterGui`.
4. Enjoy!
